Introduction

Please see here for the full documentation for renderTiles and here for the demo of basic tile rendering.

In this demo we show how basic tile rendering can be used in a google map overlay to render vector tiles on a google map. We are using a single vector source at a single zoom level.

Tiles for Brighton, UK were obtained free from openmaptiles.com for use, here, in this open source project (see readme for more details).
